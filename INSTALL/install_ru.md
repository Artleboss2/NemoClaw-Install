# Как установить NVIDIA NemoClaw: Пошаговое руководство

NemoClaw — это стек с открытым исходным кодом от NVIDIA, предназначенный для безопасного запуска ИИ-агентов OpenClaw путем добавления контроля конфиденциальности, песочницы (sandboxing) и барьеров безопасности.

Выполните эти простые шаги, чтобы настроить и запустить вашего безопасного ИИ-агента.

## Предварительные требования

Перед началом убедитесь, что ваша система соответствует следующим требованиям:

* **Операционная система:** Linux (рекомендуется Ubuntu 22.04+). macOS и Windows (через WSL2) поддерживаются, но в экспериментальном режиме.
* **Оборудование:** Не менее 8 ГБ оперативной памяти (настоятельно рекомендуется 16 ГБ) и около 40 ГБ свободного дискового пространства.
* **API-ключ:** API-ключ NVIDIA (вы можете получить его бесплатно на `build.nvidia.com`).

## Шаг 1: Подготовка вашей системы

Для правильной работы NemoClaw требуется установить Docker и Node.js (версия 20 или 22).

### 1. Проверка Docker:

Убедитесь, что Docker установлен и у вас есть необходимые права доступа, чтобы не использовать `sudo` каждый раз:

```bash
docker ps
```

(Если вы получаете ошибку отказа в доступе "permission denied", выполните `sudo usermod -aG docker $USER`, затем выйдите из системы и войдите снова).

### 2. Установка Node.js:

Выполните следующие команды для установки Node.js (версия 22):

```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
```

Проверьте установку, выполнив `node --version`.

## Шаг 2: Установка NemoClaw

NVIDIA предоставляет простой установочный скрипт, который берет на себя всю сложную работу.

Выполните следующую команду в вашем терминале:

```bash
curl -fsSL https://nvidia.com/nemoclaw.sh | bash
```

Примечание: В зависимости от конфигурации вашей системы, вам может потребоваться выполнить ее с помощью `sudo bash`, если вы столкнетесь с ошибками прав доступа.

## Шаг 3: Настройка вашего агента (Onboarding)

После завершения установки вам необходимо настроить песочницу (sandbox) и параметры вывода (inference).

Запустите мастер настройки:

```bash
nemoclaw onboard
```

Во время этого интерактивного процесса:

* Дайте имя вашему помощнику (например, `my-assistant`).
* Введите ваш API-ключ NVIDIA при появлении соответствующего запроса.
* Примите конфигурации по умолчанию для политик безопасности и настройки песочницы.

Примечание: Скрипт скачает образ песочницы (около 2,4 ГБ). Это может занять несколько минут.

## Шаг 4: Подключение и общение!

Ваша безопасная среда теперь готова. Пришло время подключиться к вашему изолированному агенту.

### 1. Подключение к песочнице:

Замените `my-assistant` на имя, которое вы выбрали на Шаге 3.

```bash
nemoclaw my-assistant connect
```

### 2. Запуск интерфейса чата:

Оказавшись внутри изолированной песочницы, запустите текстовый пользовательский интерфейс (TUI) OpenClaw:

```bash
openclaw tui
```

Поздравляем! Теперь вы можете безопасно вводить сообщения и взаимодействовать с вашим изолированным ИИ-агентом.
